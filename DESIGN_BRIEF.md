# atlas — Design Brief

> このドキュメントは atlas プロジェクトの **デザイン判断における唯一の根拠** です。
> 実装中に「色 / フォント / レイアウト / トーン」で迷ったら、ここに戻ってきて確認してください。

## 1. プロジェクト概要

- **名称**: atlas
- **読み**: アトラス
- **語義**: 地図帳・図鑑
- **コンセプト**: コーヒーで世界を旅するように、自分のテイスティング体験を 1 冊の地図帳に綴じる
- **一行で**: 「コーヒーは世界を旅する。あなたのカップで、もう一度。」

## 2. プロダクト本質

- 飲んだコーヒーの **テイスティングカルテ** を 1 杯 1 ページで残す
- 産地 / ロースター / 抽出方法 / 焙煎度 / 味のメモを記録
- **SCA フレーバーホイール** 上にタップで風味を可視化
- 各カルテは個別 URL を持ち、SNS にそのまま貼って美しいプレビューが出る
- 将来 SNS 化 → フォロー、産地別ランキング、他人のテイスティング閲覧

## 3. 想定ユーザー

- スペシャルティコーヒーに関心のある国内ユーザー
- コーヒー愛好家、自宅で淹れる人、カフェ巡り好き、バリスタ
- 「飲んだコーヒーを覚えておきたい」「テイスティングを学びたい」層
- 既存 iOS アプリ(コーヒーノート/手帳)で物足りなさを感じてる層

## 4. 既存との差別化（実調査済）

| 軸 | 既存 (Beanconqueror / コーヒーノート他) | atlas |
|---|---|---|
| プラットフォーム | iOS / Android ネイティブのみ | **Web 前提**、URL 共有可 |
| 味の記録 | 5段階スター or 数値スライダー | **SCA フレーバーホイール SVG** で視覚化 |
| 公開・共有 | アプリ内に閉じる | **個別カルテ URL** + 動的 OGP |
| 世界観 | 機能優先・素っ気ない UI | **地図帳/旅日記の世界観** |
| 産地表現 | テキストのみ | **世界地図 + 産地ピン + スタンプ** |

## 5. 世界観 / メタファー

- **大元の比喩**: 古地図帳 (atlas)、紀行文、コーヒー商隊のジャーナル
- **連想語**: アトラス、海図、コンパスローズ、スタンプ、麻袋、コーヒーチェリー、産地のラベル、旅日記
- **避けるべき**: モダン SaaS テンプレ感、Web3 風グラデ、ガラスモーフィズム、ネオン

## 6. カラーパレット

### Light モード (基本)

| ロール | 値 | 用途 |
|---|---|---|
| Background (古紙アイボリー) | `oklch(0.97 0.012 80)` ≒ `#F7F2E8` | ページ全体の背景 |
| Foreground (ダークインク) | `oklch(0.22 0.02 60)` ≒ `#2A2118` | 本文テキスト |
| Primary (焙煎ブラウン) | `oklch(0.38 0.08 50)` ≒ `#5C3A21` | ボタン、リンク、強調 |
| Accent (コーヒーチェリー赤) | `oklch(0.50 0.15 25)` ≒ `#9B3A2A` | 重要ハイライト、産地ピン |
| Secondary (生豆グリーン) | `oklch(0.55 0.07 110)` ≒ `#7C8559` | サブ要素、ラベル |
| Muted (淡カフェオレ) | `oklch(0.92 0.012 70)` ≒ `#EBE3D4` | カード背景、罫線下地 |
| Border (薄インク) | `oklch(0.85 0.015 70)` ≒ `#D5C9B5` | 区切り線、入力枠 |

### 焙煎度カラースケール

豆カードや産地ラベルで、焙煎度を色で直感的に表現する：

- Light: `oklch(0.65 0.08 60)` (キャラメル色)
- Medium: `oklch(0.50 0.09 50)` (チョコ色)
- Medium-Dark: `oklch(0.38 0.08 45)` (深いブラウン)
- Dark: `oklch(0.28 0.05 40)` (ほぼ黒)

## 7. タイポグラフィ

| 役割 | フォント | 用途 |
|---|---|---|
| 見出し (Heading) | **Fraunces** (Google Fonts、可変) | h1/h2 のセリフで「読み物」感 |
| 本文 (Body) | **Inter** + **Noto Sans JP** | 読みやすさ最優先 |
| アクセント (Handwriting) | **Caveat** | 「旅メモ」風の手書きアクセント、引用、ラベル |
| 数値 (Tabular) | **Inter** (tabular-nums) | テイスティングスコア、抽出温度等 |

サイズ階層: 12 / 14 / 16 / 18 / 24 / 32 / 48 / 64

## 8. モチーフ / 視覚的記号

- **世界地図 (古地図風)**: 産地マップに使う、ホームの装飾にも
- **コンパスローズ**: ローディング / 空状態 / フッター装飾
- **コーヒーチェリーの実**: 産地ラベル、フレーバー選択時のマーカー
- **麻袋 (jute sack)**: コーヒー豆の象徴、カードの装飾
- **パスポートスタンプ**: 産地訪問記録の表現
- **海図の罫線**: セクション区切り、表の縁取り

## 9. 手描き要素 (必ず実装する)

[[feedback-design-promise-delivery-gap]] の反省を踏まえ、**初版から確実に含める**：

1. **rough.js**: カルテの枠線、見出し下のラフな下線、ボタンの輪郭にゆらぎ
2. **Excalidraw 自前 SVG**: コーヒーカップ、麻袋、コーヒーチェリー、コンパス、地球儀 — 最低 5 点
3. **Caveat 手書きフォント**: ラベル、引用、ティップス、空状態の文言
4. **古紙テクスチャ**: SVG ノイズパターンで背景に微妙な紙感

## 10. 参考トーン（リファレンス）

- **Monocle Magazine** — 旅 × ライフスタイル雑誌の知的な版面
- **Kinfolk** — 余白と上質な写真の生活感
- **Field Notes Brand** — 手帳的なクラフト感
- **登山地形図 / 古地図** — 産地マップの参考
- **Apple "Around the World" 系コンテンツ** — 旅の重厚感

## 11. トーン & ボイス（コピー）

- 紀行文・旅日記の温度。説明書ではない
- 「あなた」「私たち」を使わない。淡々と、しかし詩的に
- 例:
  - ✓ 「エチオピアの朝、カップから昇る花の香り。」
  - ✓ 「今日のコーヒーは、どこから来ましたか。」
  - ✗ 「ぜひあなたのテイスティングを記録してください！」
  - ✗ 「シェアして友達を増やそう！」

## 12. アンチパターン (やらないこと)

- 絵文字をアイコン代わりに使う(必ず Lucide / Phosphor)
- グラデーション乱用
- ネオン、ガラスモーフィズム、Web3 風表現
- 採用 LP 的な売り込み口調
- Slack / Discord 風のチャット UI
- ヒロセクションに「今すぐ始める」「無料登録」を巨大表示する典型 SaaS LP
